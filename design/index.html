Innovative product approach

By conducting workshops with decision-makers and management, significant breakthroughs in the work process were facilitated. The aim was to create innovative concepts that approach security holistically and transform the product into a flexible platform. This was achieved through the use of structured methods to gather observations, challenge cognitive fixedness, and generate creative solutions.

Query Builder

To improve the user experience and address confusion from the previous query system, a new and user-friendly query builder was introduced. This involved implementing a data categorization model to enhance user navigation through extensive data. The focus was on simplicity and ease of use in the query builder, empowering users to efficiently explore the system's vast amount of data without feeling overwhelmed.
